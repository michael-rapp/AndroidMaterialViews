About

"AndroidMaterialViews" is an Android-library, which provides various views, which are suggested by Android's Material Design guidelines, but which are not provided by the official Android SDK. Since the library's views are designed according to the Material Design guidelines (even on pre-Lollipop devices), the AppCompat v7 revision 21 support library or a newer one is required to be used together with the library.

The aim of this documentation is to offer an overview of the project's source code and to explain, how the interface, which is provided by the library, can be used and how the underlying implementations are supposed to work.

The library provides the following views:

  • A CircularProgressBar, which visualizes an indeterminate progress bar as an animated circle.
  • A FloatingActionButton, which is a round, colored button with a shadow and an icon. The button supports to become shown or hidden in an animated manner.

Download

The project homepage of the project "AndroidMaterialViews" is available on Sourceforge via the internet address https://sourceforge.net/p/androidmaterialviews.

The latest release of the project can be downloaded as a zip archive from the download section of the Sourceforge project site mentioned above, which is available via the direct link https://sourceforge.net/p/androidmaterialviews/files.

As well, the complete source code and documentation is available via a Mercurial repository, which can be accessed by the URL http://hg.code.sf.net/p/androidmaterialviews/code.

Example app

An Android app, which implements a use case, using this library, can be found on Sourceforge via the internet address https://sourceforge.net/p/androidmaterialviewsexample. The source code of the example App is available as a Mercurial repository, which can be accesed by the URL ssh://hg.code.sf.net/p/androidmaterialviewsexample/code. Alternativly, the latest version can be downloaded via the direct link https://sourceforge.net/p/androidmaterialviewsexample/files as a zip archive, also containing the App as a installable APK file.
